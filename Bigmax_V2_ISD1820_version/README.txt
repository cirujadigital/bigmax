# Summary

This is the second version of the Bigmax comunnicator (first version can be found here: https://www.thingiverse.com/thing:2566778) , intented to be used with a single button (play function). The record button is placed at one side of the box. The purpose of this device is to be an aid in speech stimulation therapies. This version works with an ISD1820 module, bought from aliexpress (aprox size 39x43mm)

Adittionaly You'll need:
AAA Battery x2 Holder
AAA x2 Batteries
M3 screws x 6 
Two M3 nuts
ISD1820 Module 
12x12mm soft touch switch
40-60mm diameter speaker
Wire, soldering iron,etc.

NOTE: You'll need to glue the part 'Portapilas' to the 'Piso' Part 
I recommend you to print the "caja" part rotating it on X axis at 194ยบ

Any tip is welcome! ;)
